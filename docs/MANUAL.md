# 使い方(ユーザー向けマニュアル)

「世界時計 - 世界の時間と時差」(time-j.net)の使い方です。

## 都市の時刻を調べる

目的の都市には次のいずれかからたどり着けます。

- **検索**(ヘッダーの「検索」): 日本語名(パリ)、ひらがな(ぱり)、英語名の一部(new_york)、国名(フランス)で検索できます。
- **都市一覧**: 地域ごとの50音順一覧。
- **国一覧**: 国名の50音順一覧から、その国のページへ。
- **トップページ**: 主要都市の時計、地域(アジア、ヨーロッパ…)別の一覧、人気の旅行先へのリンク。

### 都市ページで見られるもの

- **現在時刻**: 秒単位で自動更新されます。サマータイム期間中は赤く「サマータイム中」と表示されます。
- **タイムゾーン情報**: タイムゾーンの略称と日本語名、協定世界時(UTC)との時差、日本時間(JST)との時差。
- **サマータイム情報**: 実施の有無と、次にサマータイムが開始/終了する日時。
- **日本時間との変換**: 日時を選んで「変換」を押すと、現地時間⇔日本時間を相互に換算します。サマータイムも自動で考慮されます。
- **時差早見表**: 現地時間・日本時間・UTC の24時間対応表。色付きのセルは業務時間(9〜18時)の目安です。サマータイムの切替が近い場合は、切替後の表も併せて表示されます。
- **日の出・日の入りの時間**: 今日の日の出・日の入り・昼の長さを表示します。日付を変えて計算することもできます(旅行や撮影の計画に)。高緯度の白夜・極夜にも対応しています。
- **気候**: 月別の平均気温・降水量などの平年値(データのある都市のみ)。

## 国のページ

国名からたどると、その国の時刻のページが開きます。

- アメリカやロシアのように**複数のタイムゾーンがある国**では、ゾーンごとの現在時刻と主な都市の一覧を表示します。
- **EU** のページでは加盟27カ国の現在時刻を一覧できます。
- シンガポールなどの都市国家は、都市のページがそのまま国のページです。

## ツール

- **世界の会議時間の計算**(/WorldTime/Meeting): 日付と都市(最大4つ)を選ぶと、それぞれの都市の対応する時刻の24時間分の一覧表を表示します。色付きのセルが業務時間(9〜18時)なので、全員の業務時間が重なる時間帯がひと目で分かります。海外との会議・オンライン通話・ゲームの約束などにどうぞ。
- **カウントダウン ニューイヤー**(/WorldTime/Home/NewYear): 世界の主要都市が新年を迎えるまでの残り時間と、初日の出の時刻・初日の出までの残り時間を表示します。世界で一番早く新年を迎えるキリバスのライン諸島・サモアから、一番遅いアメリカ領サモアまで並んでいます。

## よくある質問

**Q. 時刻は正確ですか?**
お使いの端末の時計を基準に表示しています。端末の時計が合っていれば正確です(スマートフォンや PC は通常自動で時刻合わせされています)。

**Q. サマータイムはどう扱われますか?**
IANA タイムゾーンデータベースに基づいて自動で判定します。時刻の表示・変換・早見表のすべてでサマータイムを考慮しています。各国の制度変更にはブラウザの更新を通じて自動で追随します。

**Q. 日の出・日の入りの時刻はどのくらい正確ですか?**
天文計算(NOAA のアルゴリズム)による理論値で、誤差は数分程度です。同じ都市でも場所や標高、地形によって実際に太陽が見える時刻は変わりますので、目安としてご利用ください。

**Q. JavaScript を切っていると使えますか?**
時刻の表示・変換・検索には JavaScript が必要です。都市の基本情報(タイムゾーン、標準時の UTC との時差など)は JavaScript なしでも表示されます。

**Q. 昔の URL のままブックマークしていますが使えますか?**
使えます。ページの URL 構造は旧サイトから変わっていません。廃止した一部のページ(ブログ等)を除き、旧 URL からは自動で転送されます。
